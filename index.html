Measurement architecture测度架构

A regulator-grounded scoring pipeline for long-form disclosures 面向长篇披露文本的监管框架化评分流程

The site now follows the manuscript workflow: the PBOC maturity framework defines the scoring grid; eight domain-specific scoring agents read the source report; an evaluation layer checks source support and score alignment; accepted item scores are standardized into domain and total indices. 网页按照论文流程图重排：央行成熟度框架定义评分网格；八个分领域评分智能体读取报告原文；评估层检查原文依据与分值匹配；通过后的分项得分再标准化为能力域指数和总指数。

Core clarification 核心说明 Measurement validity测度有效性

The index is tied to a formal maturity grid with item definitions, tiered standards, and domain-level aggregation rules.指数锚定正式成熟度指标体系，包含能力项定义、档位标准和能力域汇总规则。

Core clarification 核心说明 Process documentation流程说明

The workflow records source files, request identifiers, item scores, and scoring rationales at the report level.流程在报告层面保留来源文件、请求编号、分项得分和评分理由。

Core clarification 核心说明 Benchmarking and validation基准与验证

Manual scoring, patent outcomes, keyword measures, and the digital renminbi pilot are used as complementary checks.人工评分、金融科技专利、关键词法和数字人民币试点共同作为补充检验。

Guidelines on Financial Digital Capability Maturity《金融数字化能力成熟度指引》

8domains能力域 24subdomains能力子域 75items能力项

Each capability item is scored on a 0-5 maturity scale. Domain scores are normalized by the maximum possible item score, which keeps domains comparable despite different item counts. 每个能力项按照 0-5 的成熟度档位评分。能力域得分按本域最高可能分标准化，从而保持跨领域可比性。

Scoring modules评分模块

Eight domain agents, one common scoring contract 八个领域智能体与统一评分约束

The eight agents form a coordinated division of labor under the same 75-item maturity framework. Each agent is responsible for a coherent domain and the same output contract: item score, source-grounded rationale, and structured result fields. 八个智能体是在同一个 75 项成熟度框架下形成的分工体系。每个智能体负责一个能力域，并遵循同一输出约束：分项得分、基于原文的理由和结构化结果字段。

Common scoring contract统一评分约束

All agents use the same model setting and output structure: Qwen-Long, temperature 0, item scores from 0 to 5, source-grounded rationale, report identifier, and request identifier. Full prompt files are provided as downloadable method materials.所有智能体使用同一模型设置和输出结构：通义千问长文本模型、温度参数设为零、分项评分为零至五分、评分理由基于原文，并保留报告标识和请求编号。完整提示词作为方法材料下载提供。

Validity evidence有效性证据

Checks aligned with the main measurement concerns围绕主要测度关切的检验

The website emphasizes the validation exercises that speak directly to measurement quality: human-coded benchmarks, external non-textual outcomes, alternative text measures, and a policy shock tied to digital infrastructure and payment capability. 网页重点展示与测度质量直接相关的检验：人工编码基准、外部非文本结果、替代文本测度，以及与数字基础设施和支付能力相关的政策冲击。

92.5% manual benchmark accuracy人工基准准确率

Six researchers independently scored 200 outputs; the system is evaluated against the modal human benchmark.六名研究人员独立评分 200 个输出，以众数人类评分作为基准比较系统结果。

The fintech patent validation sample covers 41 banks and 384 bank-year observations.金融科技专利验证样本覆盖 41 家银行和 384 个银行-年份观测。
